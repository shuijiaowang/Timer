# Timer
倒计时，定时，循环，队列，组合

最基础的就是一个提醒功能，什么提醒呢，时间到了就提醒
1.如喝水打卡，每一小时提醒一次的，这种是规律的，快速提醒一下然后自动取消，不会常驻，即使提醒了不喝也没问题。
2.一种是强制任务要求的，阻塞的，如需要按时吃药，必须要手动点击关闭提醒。
定时任务，一次性的，每日的，工作日的，和手机上的闹钟一样，可以随时修改，可以随时打开关闭。
倒计时任务，工作学习俩小时就结束去休息，倒计时8小时就去偷菜。
循环任务，循环倒计时任务，每1小时喝水
队列任务，先倒计时45分钟工作，在倒计时15分钟休息。
循环队列任务，先倒计时45分钟工作，在倒计时15分钟休息，再倒计时45分钟，再倒计时15分钟
定时引爆倒计时，如明天下午两点，开始番茄闹钟。
